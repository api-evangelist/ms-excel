Work IQ provides a workplace intelligence layer enabling agents to access organizational data and tools with built-in governance. The API endpoints become generally available June 16 with consumption-based pricing independent of Microsoft 365 Copilot licensing, and new enterprise governance controls are available in the Microsoft 365 admin center.
